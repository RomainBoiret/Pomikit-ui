# Storybook & playground

## Storybook

Catalogue interactif des composants (stories dans `/stories`).

```bash
pnpm storybook
```

Ouvre typiquement `http://localhost:6006`.

## Playground

App de démo locale (thèmes, formulaire, Collection, Skeleton, Toast, Dialog…).

```bash
pnpm dev
```

Ouvre typiquement `http://localhost:5173`.

## Quand utiliser quoi ?

| Outil | Usage |
| --- | --- |
| Docs (ce site) | API, patterns, intent |
| Storybook | Isolation visuelle / états |
| Playground | Composition réelle bout-en-bout |
