# Herramientas de desarrollo

## Sitio de docs (oficial)

Este sitio VitePress es la documentación **oficial**: Getting Started, Design Kits, APIs de intención y referencias de componentes. Prefiere estas páginas y el [Playground](/playground) al aprender o compartir Pomikit.

## Storybook (interno)

Storybook es un espacio de **aislamiento interno** para contribuidores:

- Regresión visual y estados de componentes
- Stories de escape hatch (`variant`, `tone`, `size`) fuera de las docs primarias
- Checks de accesibilidad e interacción en aislamiento

No es la superficie de producto. No trates las stories de Storybook como la guía canónica de API; a menudo exploran bordes avanzados que Getting Started oculta a propósito.
