# 開発ツール

## ドキュメントサイト（公式）

この VitePress サイトが**公式**ドキュメントです：はじめに、Design Kits、意図 API、コンポーネントリファレンス。Pomikit を学ぶ・共有するときは、これらのページと [プレイグラウンド](/playground) を優先してください。

## Storybook（内部）

Storybook は貢献者向けの**内部隔離**ワークスペースです：

- ビジュアル回帰とコンポーネント状態
- 主要なドキュメントから外した逃げ道ストーリー（`variant`、`tone`、`size`）
- 隔離環境でのアクセシビリティとインタラクション確認

製品の面ではありません。Storybook ストーリーを正規の API ガイドとして扱わないでください。はじめにが意図的に隠す高度な端をよく探索します。
