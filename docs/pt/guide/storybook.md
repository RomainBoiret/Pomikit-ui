# Ferramentas de desenvolvimento

## Site de docs (oficial)

Este site VitePress é a documentação **oficial**: Getting Started, Design Kits, APIs de intenção e referências de componentes. Prefira estas páginas e o [Playground](/playground) ao aprender ou compartilhar o Pomikit.

## Storybook (interno)

O Storybook é um espaço de **isolamento interno** para contribuidores:

- Regressão visual e estados de componentes
- Stories de escape hatch (`variant`, `tone`, `size`) fora das docs primárias
- Checks de acessibilidade e interação em isolamento

Não é a superfície do produto. Não trate as stories do Storybook como o guia canônico de API; elas frequentemente exploram bordas avançadas que o Getting Started esconde de propósito.
