# Outils de développement

## Site de documentation (officiel)

Ce site VitePress est la documentation **officielle** : Premiers pas, Design Kits, API d'intention et références des composants. Préférez ces pages et le [Playground](/playground) pour apprendre ou partager Pomikit.

## Storybook (interne)

Storybook est un espace d'**isolation interne** pour les contributeurs :

- Régression visuelle et états des composants
- Stories d'échappatoire (`variant`, `tone`, `size`) tenues hors de la documentation principale
- Contrôles d'accessibilité et d'interaction en isolation

Ce n'est pas la surface produit. Ne traitez pas les stories Storybook comme le guide d'API canonique ; elles explorent souvent des cas avancés que Premiers pas masque volontairement.
