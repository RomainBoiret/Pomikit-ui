# 개발 도구

## 문서 사이트 (공식)

이 VitePress 사이트가 **공식** 문서입니다: 시작하기, Design Kits, 의도 API, 컴포넌트 레퍼런스. Pomikit를 배우거나 공유할 때는 이 페이지들과 [플레이그라운드](/playground)를 우선하세요.

## Storybook (내부)

Storybook은 기여자를 위한 **내부 격리** 워크스페이스입니다:

- 시각 회귀와 컴포넌트 상태
- 주 문서에서 빼 둔 탈출구 스토리 (`variant`, `tone`, `size`)
- 격리 환경의 접근성과 인터랙션 검사

제품 표면이 아닙니다. Storybook 스토리를 정식 API 가이드로 취급하지 마세요. 시작하기가 의도적으로 감춘 고급 경계를 자주 탐색합니다.
