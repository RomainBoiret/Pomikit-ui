# Types partagés

```ts
import type { PomiSize, PomiTone, PomiVariant } from 'pomikit-ui'
```

| Type | Valeurs |
| --- | --- |
| `PomiSize` | `'sm' \| 'md' \| 'lg'` |
| `PomiVariant` | `'solid' \| 'outline' \| 'ghost' \| 'soft'` |
| `PomiTone` | `'primary' \| 'neutral' \| 'danger' \| 'success'` |

Utilisés par Button (et d’autres contrôles pour `size` / `tone`).
